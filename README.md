### Objetivo

Objetivo do projeto é o compreendimento do C# e .NET.
Para isso será desenvolvido uma API que tem a função de:
 * Cadastrar Pessoa
 * Cadastrar Produtos
 * Cadastrar Compras
 * Listar as compras de cada pessoa
